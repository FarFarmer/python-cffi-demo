# python-cffi-demo

- Inspired by Armin Ronacher's ["Beautiful Native Libraries"](http://lucumr.pocoo.org/2013/8/18/beautiful-native-libraries/)


## How to build this demo

```
mkdir build
cd build
cmake ..
make
cd ..
py.test -vv test.py
```
